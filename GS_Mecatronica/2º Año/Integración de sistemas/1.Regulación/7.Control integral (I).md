## Control integral (I)
---
El control integral elimina el error estacionario sumando el error a lo largo del tiempo.
$$u(t) = K_i \int e(t) \, dt$$
### Características
- Actúa cuando existe un error sostenido.
- Corrige el error acumulado hasta hacerlo cero.
- Si se usa solo $\rightarrow$ respuesta lenta y oscilante.
