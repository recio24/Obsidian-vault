## Control ON-OFF
---
El control **ON-OFF** es el mas simple, la salida del controlador solo puede estar encendida (ON) o apagada (OFF). El controlador compara la referencia con la variable real y aplica:
- Error > consigna = ON
- Error ≤ consigna = OFF
>[!Example] Ejemplo
>Un termostato doméstico, que enciende la calefacción cuando baja la temperatura y la apaga al alcanzar el valor deseado.

El resultado es una oscilación de la variable controlada sobre la consigna.
