# Características de los sensores
---
>[!info] Definición
>Las **características de los sensores** determinan su comportamiento, precisión y fiabilidad.  
>Describen cómo responde el sensor ante la magnitud física que mide y en qué condiciones lo hace correctamente.
### Zona de medición o rango
Es el **intervalo de valores** donde el sensor mide correctamente. Por ejemplo un sensor de temperatura que mide de -30ºC a 80ºC.
$Span = 80 - (-30) = 110^\circ C$
### Sensibilidad
Mide **cuanto cambia la señal de la salida** cuando cambia la magnitud medida.
$S = \frac{\Delta\text{salida}}{\Delta\text{entrada}}$
Ejemplo si al aumentar 1ºC la saluda sube 5mV 
$S = 5mV/^\circ C$ 
>Un sensor con sensibilidad alta y constante es el deseo de un buen sensor.
### Resolución
Es el **mínimo cambio** que el sensor puede detectar. Por ejemplo un sensor que mide fuerza de 20N a 150N y tiene una resolución del 0,1%:
$0,1\% \times (150-20) = 0,13N$
El sensor detectara cambios de 0,13N
### Precisión o exactitud
Indica que tan cerca esta el valor medio del valor real. Puede expresarse como:
- $\pm \text{valor absoluto} \rightarrow \pm 1^\circ C$
- $\% \text{del rango} \rightarrow \pm 0,5\%$
Ejemplo. [[Ejercicios. Sensorica#1. Tenemos un sensor de posición. La señal eléctrica es de 0 a +10V. Campo de medida de 0-250mm.|Ejercicio 1]]
### Offset o error del punto cero
Es el valor que muestra el sensor cuando **debería marcar 0**. Por ejemplo si un sensor de distancia con 0mm de distancia da 5mV $\rightarrow$ Offset = +5mV. Normalmente los sensores cuentan con sistema de calibración para salvar el Offset.
### Histéresis
Es el rango que diferencia el punto de encendido del de apagado. Por ejemplo un termostato de una vivienda se apaga al llegar a 27ºC y se enciende al bajar de 25ºC. La histéresis seria como un 25ºC $\pm 1$. Esto evita que se encienda y apague tan seguido.
### Repetibilidad
Es la capacidad del sensor se **dar la misma medida ante la misma lectura**. Por ejemplo si en la misma pieza mide 10.0, 10.1, 9.9 la repetibilidad es buena. 
> Un buen sensor tendrá buena precisión y buena repetibilidad, pero un sensor puede medir mal y siempre tener buena repetibilidad.
### Condiciones ambientales
Indica las condiciones físicas que el sensor puede sufrir sin dañarse, por ejemplo:
- Temperaturas de trabajo
- Protección IP
---
>[!summary] Resumen rápido
>- **Rango:** valores que puede medir.  
>- **Sensibilidad:** cuánto varía la salida.  
>- **Resolución:** cambio mínimo detectable.  
>- **Precisión:** diferencia con el valor real.  
>- **Offset:** error del cero.  
>- **Histéresis:** diferencia entre activar y desactivar.  
>- **Repetibilidad:** mide siempre igual.
